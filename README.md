# doc-to-markup
# Written Document to Markup Converter

Converts scanned/printed documents (PDF, PNG, JPEG) into clean Markdown automatically.

## Stack
- PaddleOCR — text extraction
- LayoutLMv3 — document layout detection
- FastAPI — backend API
- React — frontend UI

## Quick Start
```bash
git clone https://github.com/YOUR_USERNAME/doc-to-markup.git
cd doc-to-markup
docker compose up
```
